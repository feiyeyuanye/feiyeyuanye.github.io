项目经验

  • 窠浦
    • 项目简介：
      集团差旅管理和环球旅行定制服务平台。主要负责包括像机票、酒店、捕货，旅游线路等模块的维护，以及新模块如竞游吧、ERTC，优惠券等的开发。
    • 技能点：
      1、MVP 架构模式，结合 Retrofit 与 RxJava 做网络请求。
      2、Hybrid 结合 HightChart 的数据展示。
      3、使用 EventBus 简化 Android 事件传递。
      4、七牛实现图片的上传发布，Glide 结合 PhotoView 预览。
      5、使用 Bugly 与 LeakCanary，实现对项目的监控与一些模块与组件的优化。
  • 公卫医疗
    • 项目简介：
      便于公共卫生医疗服务中心能更快和更便捷的，采集并上传体检数据。以及通过刷卡登录，建档，多设备蓝牙连接采集体检数据，各类慢性病与婴幼儿数据采集，中医体征自测，教育宣传等。对用户从建档到信息记录查询等的全面管理。
    • 技能点：
      1、从零开始搭建基于 MVC 的架构，采用 Material Design 设计风格。
      2、蓝牙实现刷卡登录，多仪器自动连接，数据解析转换，MAC 缓存更改。
      3、自定义 View，如多项选择列表，家系图，签字板，SurfaceView 轮播图等。
      4、引入 ButterKnife 注解，OkHttp 对接 PHP 接口，fastjson 解析。
      5、布局优化，离线缓存，根据业务情况做一些编码上的性能优化。
  • 火柴盒
    • 项目简介：
      以陌生人为导向的，一款社交产品。
    • 技能点：
      1、QQ，微信，新浪第三方的注册和登录。
      2、使用环信实现通知的推送以及好友之间的聊天功能。
      3、Fragment 里嵌套 Fragment,重写 OnDestroyView() 方法解决异常。
      4、使用 PullToRefresh 第三方框架中的 ScrollView 实现下拉刷新效果。
      5、使用 XUtils，用 View 模块对布局进行注解减少代码量；用 http 模块进行网络请求；使用 BitmapUtils 模块处理图片，可无需考虑 bitmap 加载过程中出现的 OOM 和 android 容器快速滑动时出现的图片错位等现象；使用用 DBUtils 模块将网络数据保存到数据库。
