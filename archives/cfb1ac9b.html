Android еҗҜеҠЁдјҳеҢ–

еҸ‘иЎЁдәҺ 2020-06-11 жӣҙж–°дәҺ 2020-06-29 еҲҶзұ»дәҺ Android зҹҘиҜҶзӮ№ йҳ…иҜ»ж¬Ўж•°пјҡ Valineпјҡ жң¬ж–Үеӯ—ж•°пјҡ 2.1k йҳ…иҜ»ж—¶й•ҝ ≈ 2 еҲҶй’ҹ

е…«з§’е®ҡеҫӢпјҡжҳҜеңЁдә’иҒ”зҪ‘йўҶеҹҹеӯҳеңЁзҡ„дёҖдёӘе®ҡеҫӢпјҢеҚіжҢҮз”ЁжҲ·и®ҝй—®дёҖдёӘзҪ‘з«ҷж—¶пјҢеҰӮжһңзӯүеҫ…зҪ‘йЎөжү“ејҖзҡ„ж—¶й—ҙи¶…иҝҮ 8 з§’пјҢдјҡжңүи¶…иҝҮ 70% зҡ„з”ЁжҲ·ж”ҫејғзӯүеҫ…гҖӮ

еҗҜеҠЁж–№ејҸ

еҜ№дәҺи®Ўз®—жңәпјҢз®ҖеҚ•зҡ„и®ІпјҢејҖжңәжҳҜеҶ·еҗҜеҠЁпјҢйҮҚеҗҜжҳҜзғӯеҗҜеҠЁгҖӮеңЁ App дёӯпјҡ

  • еҶ·еҗҜеҠЁпјҡзі»з»ҹжІЎжңүиҜҘеә”з”Ёзҡ„иҝӣзЁӢпјҢйңҖиҰҒеҲӣе»әдёҖдёӘж–°зҡ„иҝӣзЁӢеҲҶй…Қз»ҷеә”з”ЁпјҢжүҖд»Ҙдјҡе…ҲеҲӣе»әе’ҢеҲқе§ӢеҢ–Applicationзұ»пјҢеҶҚеҲӣе»әе’ҢеҲқе§ӢеҢ–MainActivityзұ»пјҲеҢ…жӢ¬дёҖзі»еҲ—зҡ„жөӢйҮҸгҖҒеёғеұҖгҖҒз»ҳеҲ¶пјүпјҢжңҖеҗҺжҳҫзӨәеңЁз•ҢйқўдёҠгҖӮ

  • зғӯеҗҜеҠЁпјҡ зі»з»ҹдёӯе·ІжңүиҜҘеә”з”Ёзҡ„иҝӣзЁӢпјҲдҫӢпјҡжҢүbackй”®гҖҒhomeй”®пјҢеә”з”ЁиҷҪз„¶дјҡйҖҖеҮәпјҢдҪҶжҳҜиҜҘеә”з”Ёзҡ„иҝӣзЁӢиҝҳжҳҜдҝқз•ҷеңЁеҗҺеҸ°пјүпјҢеҪ“еҶҚж¬ЎзӮ№ејҖAPPй©¬дёҠиғҪеӨҹжҒўеӨҚеҲ°дёҠж¬ЎдҪҝз”Ёзҡ„зҠ¶жҖҒпјҢдёҚйңҖиҰҒеҶҚеӣһеҲ°жүӢжңәзҡ„йҰ–йЎөжү“ејҖеә”з”ЁзЁӢеәҸгҖӮдёҚдјҡеҲӣе»әе’ҢеҲқе§ӢеҢ–Applicationзұ»пјҢзӣҙжҺҘеҲӣе»әе’ҢеҲқе§ӢеҢ–MainActivityзұ»пјҲеҢ…жӢ¬дёҖзі»еҲ—зҡ„жөӢйҮҸгҖҒеёғеұҖгҖҒз»ҳеҲ¶пјүпјҢжңҖеҗҺжҳҫзӨәеңЁз•ҢйқўдёҠгҖӮ

еҶ·еҗҜеҠЁжөҒзЁӢ

йҖҡиҝҮзҝ»йҳ… Application еҗҜеҠЁзҡ„жәҗз ҒпјҢеҪ“жҲ‘д»¬зӮ№еҮ»жЎҢйқўеӣҫж Үиҝӣе…ҘжҲ‘д»¬иҪҜд»¶еә”з”Ёзҡ„ж—¶еҖҷпјҢдјҡз”ұ AMS йҖҡиҝҮ Socket з»ҷ Zygote еҸ‘йҖҒдёҖдёӘ fork еӯҗиҝӣзЁӢзҡ„ж¶ҲжҒҜпјҢеҪ“ Zygote fork еӯҗиҝӣзЁӢе®ҢжҲҗд№ӢеҗҺдјҡйҖҡиҝҮеҸҚе°„еҗҜеҠЁ ActivityThread##main еҮҪж•°пјҢжңҖеҗҺеҸҲз”ұ AMS йҖҡиҝҮ aidl е‘ҠиҜү ActivityThread##H жқҘеҸҚе°„еҗҜеҠЁеҲӣе»әApplication е®һдҫӢпјҢе№¶дё”дҫқж¬Ўжү§иЎҢ attachBaseContext гҖҒonCreate з”ҹе‘Ҫе‘ЁжңҹпјҢз”ұжӯӨеҸҜи§ҒжҲ‘д»¬дёҚиғҪеңЁиҝҷ 2 дёӘз”ҹе‘Ҫе‘ЁжңҹйҮҢеҒҡдё»зәҝзЁӢиҖ—ж—¶ж“ҚдҪңгҖӮ

жҖ»з»“пјҡApplication жһ„йҖ ж–№жі• вҖ“> attachBaseContext() вҖ“> onCreate() вҖ“> Activity жһ„йҖ ж–№жі• вҖ“> onCreate() вҖ“> й…ҚзҪ®дё»йўҳдёӯиғҢжҷҜзӯүеұһжҖ§ вҖ“> onStart() вҖ“> onResume() вҖ“> жөӢйҮҸеёғеұҖз»ҳеҲ¶жҳҫзӨәеңЁз•ҢйқўдёҠгҖӮ

  • еҲӣе»әе’ҢеҲқе§ӢеҢ–Applicationзұ»гҖҒеҲӣе»әMainActivityгҖӮ

  • inflateеёғеұҖгҖҒеҪ“onCreate/onStart/onResumeж–№жі•йғҪиө°е®ҢгҖӮ

  • contentView зҡ„ measure/layout/draw жҳҫзӨәеңЁз•ҢйқўдёҠгҖӮ

дјҳеҢ–ж–№ејҸ

зЁӢеәҸеңЁеҶ·еҗҜеҠЁзҡ„ж—¶еҖҷпјҢдјҡжңү 1s е·ҰеҸізҡ„зҷҪеұҸй—ӘзҺ°пјҢдҪҺзүҲжң¬жҳҜй»‘еұҸзҡ„зҺ°иұЎгҖӮapp еҗҜеҠЁеҗҺпјҢWindowManager дјҡе…ҲеҠ иҪҪ app theme дёӯзҡ„ windowBackgroundгҖӮ

еёёи§Ғзҡ„и§ЈеҶіж–№жЎҲеҰӮпјҡ

  • йҖҡиҝҮи®ҫзҪ® windowIsTranslucent йҖҸжҳҺеұһжҖ§пјҢиҷҪз„¶жІЎжңүдәҶзҷҪеұҸпјҢдҪҶжҳҜдёӯй—ҙиҝҳжҳҜжңүдёҖе°Ҹж®өдёҚеҸҜи§ҒпјҢзӮ№еҮ»еә”з”Ёеӣҫж Үдјҡжңүз§ҚеҚЎйЎҝзҡ„ж„ҹи§үпјҢе»¶иҝҹдёҖз§’жүҚжү“ејҖеә”з”ЁгҖӮ
  • дёҖиҲ¬еә”з”ЁйғҪдјҡжңүдёӘеҗҜеҠЁйЎөпјҢеҸҜд»ҘеңЁзҷҪеұҸжңҹй—ҙпјҢж”ҫдёҖеј еҗҢеҗҜеҠЁйЎөдёҖж ·зҡ„еӣҫзүҮиҝҮжёЎпјҢдҪҶжіЁж„ҸеӨ„зҗҶзҠ¶жҖҒж ҸгҖӮ
  • дҪҝз”Ё Splash зҡ„е№ҝе‘ҠйЎөпјҢеҗҢж—¶еҶҚеўһеҠ дёҖдёӘеҖ’ж•°зҡ„и®Ўж—¶еҷЁпјҢжңҖеҗҺжүҚиҝӣе…ҘеҲ°зҷ»еҪ•йЎөйқўжҲ–иҖ…дё»йЎөйқўгҖӮе№ҝе‘ҠйЎөеҸҜд»ҘжҳҜз¬¬дёүж–№пјҢд№ҹеҸҜд»ҘжҳҜеҜ№иҝҷдёӘйЎ№зӣ®зҡ„з®Җд»ӢпјҢиҖҢдё”еңЁеҖ’ж•°зҡ„ж—¶еҖҷеҸҜд»ҘеҜ№дёҖдәӣжҸ’д»¶е’Ңеҝ…йЎ»жҲ–иҖ…иҖ—ж—¶зҡ„еҲқе§ӢеҢ–еҒҡдёҖдәӣеҮҶеӨҮгҖӮ

зҹҘйҒ“дәҶ attachBaseContext гҖҒonCreate еңЁеә”з”ЁдёӯжңҖе…ҲеҗҜеҠЁпјҢйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»ҘйҖҡиҝҮ TreceView зӯүжҖ§иғҪжЈҖжөӢе·Ҙе…·пјҢжқҘжЈҖжөӢе…·дҪ“еҮҪж•°иҖ—ж—¶ж—¶й—ҙпјҢз„¶еҗҺжқҘеҜ№е…¶еҒҡе…·дҪ“зҡ„дјҳеҢ–гҖӮ

  • йЎ№зӣ®дёҚеҸҠж—¶йңҖиҰҒзҡ„д»Јз ҒйҖҡиҝҮејӮжӯҘеҠ иҪҪгҖӮ
  • е°ҶеҜ№дёҖдәӣдҪҝз”ЁзҺҮдёҚй«ҳзҡ„еҲқе§ӢеҢ–пјҢеҒҡжҮ’еҠ иҪҪгҖӮ
  • е°ҶеҜ№дёҖдәӣиҖ—ж—¶д»»еҠЎйҖҡиҝҮејҖеҗҜдёҖдёӘ IntentService жқҘеӨ„зҗҶгҖӮ
  • иҝҳйҖҡиҝҮ redex йҮҚжҺ’еҲ— class ж–Үд»¶пјҢе°ҶеҗҜеҠЁйҳ¶ж®өйңҖиҰҒз”ЁеҲ°зҡ„ж–Үд»¶еңЁ APK ж–Үд»¶дёӯжҺ’еёғеңЁдёҖиө·пјҢе°ҪеҸҜиғҪзҡ„еҲ©з”Ё Linux ж–Үд»¶зі»з»ҹзҡ„ pagecache жңәеҲ¶пјҢз”ЁжңҖе°‘зҡ„зЈҒзӣҳ IO ж¬Ўж•°пјҢиҜ»еҸ–е°ҪеҸҜиғҪеӨҡзҡ„еҗҜеҠЁйҳ¶ж®өйңҖиҰҒзҡ„ж–Үд»¶пјҢеҮҸе°‘ IO ејҖй”ҖпјҢд»ҺиҖҢиҫҫеҲ°жҸҗеҚҮеҗҜеҠЁжҖ§иғҪзҡ„зӣ®зҡ„гҖӮ
  • 5.0 дҪҺзүҲжң¬еҸҜд»ҘеҒҡ MultiDex дјҳеҢ–пјҢеңЁз¬¬дёҖж¬ЎеҗҜеҠЁзҡ„ж—¶еҖҷпјҢзӣҙжҺҘеҠ иҪҪжІЎжңүз»ҸиҝҮ OPT дјҳеҢ–зҡ„еҺҹе§Ӣ DEXпјҢе…ҲдҪҝеҫ— APP иғҪеӨҹжӯЈеёёеҗҜеҠЁгҖӮз„¶еҗҺеңЁеҗҺеҸ°еҗҜеҠЁдёҖдёӘеҚ•зӢ¬иҝӣзЁӢпјҢж…ўж…ўең°еҒҡе®Ң DEX зҡ„ OPT е·ҘдҪңпјҢе°ҪеҸҜиғҪйҒҝе…ҚеҪұе“ҚеҲ°еүҚеҸ° APP зҡ„жӯЈеёёдҪҝз”ЁгҖӮ

Application еҗҜеҠЁе®Ңд№ӢеҗҺпјҢAMS дјҡжүҫеҮәеүҚеҸ°ж ҲйЎ¶еҫ…еҗҜеҠЁзҡ„ Activity , жңҖеҗҺд№ҹжҳҜйҖҡиҝҮ AIDL йҖҡзҹҘ ActivityThread#H жқҘиҝӣиЎҢеҜ№ Activity зҡ„е®һдҫӢеҢ–е№¶дҫқж¬Ўжү§иЎҢз”ҹе‘Ҫе‘Ёжңҹ onCreateгҖҒonStartгҖҒonRemuse еҮҪж•°пјҢйӮЈд№ҲиҝҷйҮҢз”ұдәҺ onCreate з”ҹе‘Ҫе‘ЁжңҹдёӯеҰӮжһңи°ғз”ЁдәҶ setContentView еҮҪж•°пјҢеә•еұӮе°ұдјҡйҖҡиҝҮе°Ҷ XML2View йӮЈд№ҲиҝҷдёӘиҝҮзЁӢиӮҜе®ҡжҳҜиҖ—ж—¶зҡ„гҖӮ

жүҖд»ҘиҰҒзІҫз®Җ XML еёғеұҖд»Јз ҒпјҢе°ҪеҸҜиғҪзҡ„дҪҝз”Ё ViewStubгҖҒinclude гҖҒmerge ж ҮзӯҫжқҘдјҳеҢ–еёғеұҖгҖӮжҺҘзқҖеңЁ onResume еЈ°жҳҺе‘ЁжңҹдёӯдјҡиҜ·жұӮ JNI жҺҘж”¶ Vsync (еһӮзӣҙеҗҢжӯҘеҲ·ж–°зҡ„дҝЎеҸ·) иҜ·жұӮпјҢ16ms д№ӢеҗҺеҰӮжһңжҺҘж”¶еҲ°дәҶеҲ·ж–°зҡ„ж¶ҲжҒҜпјҢйӮЈд№Ҳе°ұдјҡеҜ№ DecorView иҝӣиЎҢ onMeasure->onLayout->onDraw з»ҳеҲ¶гҖӮжңҖеҗҺжүҚжҳҜе°Ҷ Activity зҡ„ж №еёғеұҖ DecorView ж·»еҠ еҲ° Window е№¶дәӨдәҺ SurfaceFlinger жҳҫзӨәгҖӮ

жүҖд»ҘиҝҷдёҖжӯҘйҷӨдәҶиҰҒзІҫз®Җ XML еёғеұҖпјҢиҝҳжңүеҜ№иҮӘе®ҡд№ү View зҡ„жөӢйҮҸпјҢеёғеұҖпјҢз»ҳеҲ¶зӯүеҮҪж•°дёҚиғҪжңүиҖ—ж—¶е’ҢеҜјиҮҙ GC зҡ„ж“ҚдҪңгҖӮжңҖеҗҺд№ҹеҸҜд»ҘйҖҡиҝҮ TreaceView е·Ҙе…·жқҘжЈҖжөӢиҝҷдёүдёӘеЈ°жҳҺе‘ЁжңҹиҖ—ж—¶ж—¶й—ҙпјҢд»ҺиҖҢиҝӣдёҖжӯҘдјҳеҢ–пјҢиҫҫеҲ°жһҒйҷҗгҖӮ
