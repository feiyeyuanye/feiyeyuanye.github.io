其它

解决端口被占用的问题：Port 4000 has been used. Try other port instead.

方法一：hexo s -p 5000

查询 SSH 的连接状态：ssh -T git@github.com

Github 只支持公开仓库用来做静态博客，私有仓库不可以（开会员好像可以）。

将 Github 的公开仓库切换为私有仓库后，页面无法打开，再切回公开的也不行，可能是需要重新配置下 SSH。

关于证书，当解析地址和提供的证书不一致时，会出问题。（比如博客是双地部署，国内优先使用 Coding，证书也应该是 Coding 提供的，如果证书是 GitHub 的，就会出问题。）。有时网站打不开，发现证书有问题时，很可能是 Coding 的证书没有续签成功，到域名解析里，先把 Github 的解析暂停，然后重新申请 Coding 的证书，当成功后再开启 GitHub 的解析。
